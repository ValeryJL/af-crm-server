# AF-CRM Server 🚀

A professional backend solution for managing technical service teams, task scheduling, and resource allocation. Designed to streamline operations for technical service providers.

## 🛠️ Tech Stack
* **Framework:** Spring Boot 3.x
* **Language:** Java 21 (LTS)
* **Database:** PostgreSQL
* **Persistence:** Spring Data JPA / Hibernate
* **Build Tool:** Maven

## 📋 Key Features (Planned)
- **Team Management:** CRUD operations for technical teams.
- **Service Scheduling:** Weekly calendar integration for service assignment.
- **Task Tracking:** Status updates and reporting for ongoing technical tasks.
- **RESTful API:** Clean and documented endpoints for frontend consumption.
